Relatório de Mapa de Calor e Distribuição Territorial

Destinatário sugerido: Ministério do Empreendedorismo | Base analisada: inscrições georreferenciadas | Data de geração: 29/06/2026

Executive Summary

  • A base tratada está pronta para análise territorial. Dos 2.609 registros recebidos, 2.297 possuem latitude e longitude válidas, equivalente a 88,0% da base original.
  • O mapa de calor foi construído com 2.251 registros no recorte principal. Esses pontos representam 86,3% do universo recebido e sustentam a leitura de concentração geográfica para planejamento de ações territoriais.
  • Foram identificados 46 registros fora do recorte geográfico esperado. Eles foram preservados para auditoria em planilha separada, sem contaminar o mapa e os clusters principais.
  • O maior agrupamento territorial concentra 804 registros. A clusterização oferece uma segmentação operacional para priorizar comunicação, logística, busca ativa e ações presenciais.

A concentração territorial está pronta para orientar atuação local

O mapa de calor deve ser usado como instrumento de priorização territorial, não como cadastro nominal. A versão interativa permite visualizar as áreas de maior concentração de inscrições georreferenciadas e alternar a camada de pontos para auditoria visual dos registros.

Recomendações para uso pelo Ministério

  1. Usar o mapa de calor para priorização territorial. Iniciar ações, visitas, parcerias ou campanhas digitais pelas regiões de maior densidade.
  2. Usar os clusters como unidades operacionais. Cada cluster pode orientar rotas, agendas, mutirões de atendimento, eventos de capacitação e comunicação regionalizada.
  3. Validar os registros fora do recorte antes de descartá-los. Parte pode representar erro de geocodificação; outra parte pode indicar inscrições realmente fora da região foco.
  4. Compartilhar preferencialmente a base anonimizada. Para circulação institucional, utilizar o arquivo base_tratada_para_ministerio.xlsx, que remove nome, e-mail, telefone e endereço completo.

Pontos que ainda merecem validação

Antes de usar a base para decisões finais de atendimento nominal, recomenda-se validar CEPs sem coordenadas, revisar os 46 registros fora do recorte e confirmar se o território de interesse é apenas Grande São Paulo ou se deve incluir outros municípios/estados.

Caveats e premissas

Geocodificação por CEP aproxima localização. As coordenadas representam uma referência geográfica associada ao CEP/endereço, não necessariamente o ponto exato de residência.

Dados pessoais foram removidos da base recomendada para entrega. O arquivo completo técnico permanece no projeto, mas a versão de entrega reduz risco de exposição indevida.

O mapa HTML depende de bibliotecas de mapa carregadas pela internet. Com conexão ativa, a camada de OpenStreetMap e o mapa de calor são renderizados normalmente no navegador.

Arquivos de referência: base_clusterizada.xlsx como base técnica completa; base_tratada_para_ministerio.xlsx como base recomendada para compartilhamento; relatorio_ministerio_empreendedorismo.html como relatório formal.
